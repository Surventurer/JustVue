## Usage

1. Enter a title for your code snippet
2. Paste or type your code in the code area
3. Click "Add Code" or press `Ctrl/Cmd + Enter`
4. Use the search bar to filter snippets by title
5. Click "Copy to Clipboard" to copy any snippet
6. Click "Delete" to remove a snippet
